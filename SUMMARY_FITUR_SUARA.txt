## ⚙️ CUSTOMIZATION

### Mengubah Kecepatan Suara

Edit **deteksi.py** atau **app.py** (line ~10):

```python
# Lambat (natural) - 120
tts_engine.setProperty('rate', 120)

# Normal - 150 (default)
tts_engine.setProperty('rate', 150)

# Cepat - 200+
tts_engine.setProperty('rate', 250)
```

### Mengubah Volume

```python
# Senyap
tts_engine.setProperty('volume', 0.0)

# Sedang
tts_engine.setProperty('volume', 0.5)

# Maksimal (default)
tts_engine.setProperty('volume', 1.0)
```

## 🔧 IMPLEMENTATION DETAILS

### Perbedaan Terminal vs Web Mode:

**Terminal (deteksi.py):**
- Blocking call untuk suara
- Tunggu audio selesai sebelum frame berikutnya
- Sederhana

**Web (app.py):**
- Non-blocking dengan threading
- Audio di thread terpisah
- Video tetap smooth

### Why pyttsx3?

✅ **Offline** - Tidak perlu internet  
✅ **Cross-platform** - Windows, macOS, Linux  
✅ **Fast** - No latency  
✅ **Lightweight** - Kecil & efficient  
✅ **Open-source** - Free & maintained
